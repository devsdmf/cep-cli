# CEP CLI

Find Brazilian addresses information by its zipcode (CEP) or the address line via CLI.

## Installation

```
$ npm i -g cep-cli
```

## Usage

Search for information about a zipcode (CEP):

```
$ cep 34000-000
```

The output will be:

```
Results of your search for 34000000:

Street: Nova Lima
Neighborhood: null
City: Nova Lima
State: MG

IBGE Code: 3144805
Area Code: 31

Altitude: 750.9
Latitude: -19.9930478
Longitude: -43.8485566
```
